合并视频

下载下来的视频，其实是分段的，不方面查看，所以还需要将视频给合并起来，这里用的是ffmpeg命令，最后合并的视频保存为output.mp4，存在当前路径下。

ffmpeg -f concat -safe 0 -i file.txt -c copy output.mp4
